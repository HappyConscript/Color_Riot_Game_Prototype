# Color_Riot_Game_Prototype
A collection of components for a mobile party game built C#. This prototype leverages windows forms to mimic a mobile interface and features turn timers, scoring, randomization, and custom objects.

This repository contains the following five files:

  1. frmMain.cs - A CS file containing all of the C# to create the functionality of the core game
  2. Item.cs - A CS file containing all of the C# to create the functionality of the Item object
  3. Player.cs - A CS file containing all of the C# to create the functionality of the Player object
  4. Program.cs - A CS file containing the auto-generated C# for the program
  5. ColorRiotPrototype.exe - Executable file for the game itself
